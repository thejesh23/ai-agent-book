# use-computer-while-calling · Dual Agent Architecture (Phone + Computer Use)

> **The complete code has been separated into a continuously maintained research project: [`19PINE-AI/TalkAct`](https://github.com/19PINE-AI/TalkAct)**
>
> ```bash
> git clone https://github.com/19PINE-AI/TalkAct.git
> ```

Corresponds to **Experiment 10-4 / 10-5** in Chapter 10 of *Deep Understanding of AI Agents*: The Phone Call Agent and Computer Use Agent communicate directly via WebSocket and collaborate in parallel to complete tasks that require both voice interaction and web page operations. The architecture, operation methods, and latest developments are all based on the documentation in the upstream TalkAct repository.
