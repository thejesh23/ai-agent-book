#### Open Action Space of LLM Agents

Modern LLM-based agents have brought revolutionary changes to action space design. Unlike the closed, limited action spaces of traditional RL, LLM agents exhibit unprecedented openness.

The most important innovation is incorporating **thinking as a special action** into the action space. The significance of this concept cannot be overstated. In traditional RL, agents can only execute external actions—movement, attacks, picking up items, and other operations that change the environment state. In modern LLM agents, however, **internal thinking and reasoning have become core components of the action space**.

Traditional agents select actions through simple policy queries, whereas LLM agents' action selection involves complex reasoning processes. The agent first thinks about the current state and the actions it should take, formulates a plan to decompose the goal into steps, and then executes actions based on that thinking and plan. This reasoning ability allows the agent to understand the deeper meaning of a task rather than merely matching patterns, engage in hypothetical reasoning and counterfactual thinking, break down complex problems into sub-problems, and use common sense to fill information gaps.

This internal thinking action has unique characteristics:
- **No immediate reward**: Thinking itself does not generate environmental rewards; only subsequent tool calls may yield rewards.
- **Virtually unlimited iterations**: The agent can engage in multiple rounds of deep thinking without environmental constraints.
- **Lower cost**: Compared to actual tool calls, the cost of thinking is generally small.

In traditional RL frameworks, actions with "no immediate reward and virtually unlimited iterations" are extremely difficult to train. But this is precisely the fundamental change brought by LLMs—unlike traditional MDP agents that start from scratch, LLM+RL agents stand on the shoulders of giants. The vast knowledge acquired through **pre-training** serves as a powerful prior, enabling the agent to inherit world knowledge, common-sense reasoning, and language understanding capabilities. The agent naturally understands the value of thinking, and through reinforcement learning, the LLM can further reinforce those thinking-execution paths that succeed in rollouts, thereby learning to better use internal thinking to accomplish tasks.

This **zero-shot generalization** ability allows the agent to understand and execute instructions it has never seen before, master new tasks with very few examples, and exhibit **few-shot adaptation** characteristics.

Beyond internal thinking, the action space of modern LLM agents also demonstrates openness in the **diversity of tool parameters**:
- **Natural language queries**: Search tools can accept arbitrarily complex natural language descriptions as parameters.
- **Program code**: Code interpreters can execute any Python, SQL, or other code dynamically generated by the agent.
- **Complex data structures**: API calls can construct nested JSON objects and dynamically generated query conditions.
- **Multimodal content**: The agent can generate and process image descriptions, audio transcriptions, and video analysis instructions.

This parameter-level openness means that even with only a few tools, the agent's actual action space is nearly infinite. For example, a code interpreter tool can theoretically execute any computable task, and a search tool can explore the entire information space of the internet.

Modern agents demonstrate this openness across various domains. **Web search agents** make open-ended API calls in the internet environment, constructing arbitrarily complex natural language queries. **Code execution agents** can generate and execute code of arbitrary complexity in a development environment, with each execution potentially being a novel, creative solution. **Computer use agents** not only perform predefined click operations in a graphical interface environment but also understand screen content and generate complex operation sequences.

The evolution from closed to open action spaces reflects a fundamental paradigm shift in AI agents. Traditional RL searches for optimal policies in limited spaces through extensive trial and error, while modern LLM-based agents leverage pre-trained knowledge for creative exploration in open spaces. This shift brings new opportunities: agents can handle unprecedented tasks, solve complex problems by combining basic tools, and continuously expand their capability boundaries through human interaction. It also brings new challenges: how to efficiently search in an infinite action space, how to evaluate and ensure the safety of generated actions, and how to define and optimize reward functions in an open environment.
